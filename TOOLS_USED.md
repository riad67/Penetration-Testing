# Tools Used in Penetration Test

## 1. Reconnaissance
- **Netdiscover** – Network host discovery
- **Ping** – Host availability check

## 2. Scanning & Enumeration
- **Nmap 7.95** – Port scanning, OS detection, service enumeration

## 3. Exploitation
- **Hydra** – SSH brute-force attack
- **OpenSSH Client** – Remote access

## 4. Post-Exploitation
- Standard Linux commands (`id`, `whoami`, `cat`, etc.)
